Elaboración de presupuestos

MMEX permite la configuración de un presupuesto para un año y/o mes.

Esto te permitiría comparar como lo hiciste respecto a tu presupuesto.

Los presupuestos se pueden mostrar en años naturales o años fiscales.

Use el menú Opciones para configurar el inicio del año fiscal y cualquier configuración permanente que se requiera.

La opción Ver→Presupuestos: como años fiscales también se puede usar para intercambiar entre las dos vistas

Creando nuevo presupuesto

Para configurar un presupuesto pinche con el botón derecho en el nodo del árbol 'Presupuestando' y añade un presupuesto mensual o anual.

Edición de categorías del presupuesto

Una vez añadido un año, selecciona el año para mostrar la configuración del presupuesto.

Cada categoría se puede editar haciendo doble click en la categoría seleccionada.

Esto se hace para todas las categorías posteriores.

Organizar presupuesto

Este se convierte en el presupuesto para ese año. Los meses y/o años posteriores se pueden derivar de ese año.


Para cada categoría se muestran sus totales.

Cuando se configura un presupuesto, los totales del presupuesto se pueden desactivar desde la opción del menú:

Ver→Configuración del presupuesto: sin totales

Usando los informes bajo 'Presupuestando' podrás comparar cómo gastas el dinero frente a tu presupuesto actual.
