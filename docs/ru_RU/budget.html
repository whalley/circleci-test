Бюджетирование в MMEX

Параметры бюджета

Вариант меню Вид → Бюджеты: как Финансовые Годы также могут использоваться для переключения между двумя видами.

MMEX позволяет настроить бюджет на год и/или месяц. Это позволит вам сравнить исполнение бюджета с его планированием.

Бюджетные планы могут отображаться в календарных или финансовых годах. При необходимости, используйте диалоговое окно «Параметры», чтобы установить начало финансового года и любые другие настройки.

Создание нового бюджета

Чтобы настроить планирование бюджета, щелкните правой кнопкой мыши узел дерева «Бюджетирование» и добавьте в бюджетный план месяц или год.

Редактирование категорий бюджета

После того, как год будет добавлен, выберите его в навигаторе для настройки.

Каждая категория может быть отредактирована двойным нажатием на выбранную категорию. Это делается для всех последующих категорий.

Панель настройки бюджета

Это становится бюджетом на этот год. Последующие месяцы и/или годы могут быть скопированы из этого года.

Сводные показатели по категориям отображаются для каждой основной категории.

При настройке бюджета Сводные Показатели Бюджета могут быть скрыты в меню: Вид → Настройка бюджета: Сводные Показатели Бюджета.

Бюджетные отчеты

Используя отчеты из раздела "Бюджет", можно сравнить, как вы потратили деньги по сравнению с фактическим бюджетом.
